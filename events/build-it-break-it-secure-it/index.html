ESEMÉNY AKTUÁLIS

Build it. Break it. Secure it.

Csatlakozz hozzánk csütörtökönként egy 6 alkalmas workshop-sorozatra, ahol a tanulás mellett a gyakorlati tapasztalatoké lesz a főszerep.

Sziasztok! 😊

Szervezünk egy 6 alkalmas, ingyenes workshopot azoknak, akiket érdekel a Windows 11, az Intune, az Entra ID vagy a Microsoft Defender világa.

Nem egy hagyományos előadás lesz, ahol végignézünk egy rakás diát. Inkább közösen építünk, tesztelünk, hibázunk, debugolunk, és közben megnézzük, hogyan működik egy modern Microsoft környezet a gyakorlatban.

Lesz saját Windows 11 kliens, Microsoft 365 E5 alapú lab környezet, Intune, Entra ID, Defender és sok valós példán keresztül szerzett tapasztalat.

⚠️ Fontos: az alkalmak egymásra épülnek, ezért a teljes workshop-sorozat heti rendszerességű követésére érdemes készülni.

A sorozat célja, hogy gyakorlati betekintést adjon a Microsoft endpoint management és endpoint security világába. 🚀
